# SimulaBank - Automação de APIs com Robot Framework

Projeto desenvolvido para fins educacionais com foco em automação de testes de APIs REST utilizando Robot Framework.

## Tecnologias

- Python
- Robot Framework
- RequestsLibrary
- Postman
- Django REST Framework
- Git
- GitHub

## Estrutura

```
tests/
resources/
data/
reports/
```

## Executar

```
robot tests
```
